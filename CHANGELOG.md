# Changelog

Все значительные изменения в этом проекте будут задокументированы в этом файле.

## [1.1.0] - 2026-01-05

### Добавлено
- Документация в README.md с подробным описанием функциональности
- Changelog для отслеживания изменений
- Улучшенные комментарии в коде с JSDoc
- Проверки на корректность данных в tokenManager.js
- Обработка ошибок в основных функциях

### Изменено
- Улучшена обработка ошибок в sendMessage() в chat.js
- Добавлены try-catch блоки для предотвращения падений приложения
- Оптимизирована функция saveTokens() с проверкой на корректность данных
- Улучшена обработка ошибок в маршрутах API
- Добавлены проверки на существование токенов при обновлении статуса
- Улучшена структура обработки ответов от API

### Исправлено
- Исправлена ошибка с дублированием export в routes.js
- Исправлены потенциальные утечки ресурсов при обработке страниц в браузере
- Исправлены потенциальные ошибки при обработке некорректных данных
- Исправлены проблемы с обработкой ошибок при работе с файлами

## [1.0.0] - 2026-01-05

### Добавлено
- Основной функционал прокси-сервера для Qwen API
- Поддержка OpenAI-совместимого API
- Многопользовательская система с ротацией аккаунтов
- Поддержка загрузки файлов
- Поддержка потоковой передачи ответов
- Система управления моделями и маппинга
- Система аутентификации и управления токенами
